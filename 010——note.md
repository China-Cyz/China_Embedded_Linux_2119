#### 一、一级指针变量（只存放一个地址，而且这个地址上面只能是数据）

```
回想一下：之前所谓的合法地址怎么得来的？
	定义一个合法的普通变量，然后用&取其地址即可。
	int a =10;//4个字节
	int * p = &a;
现在我们可以不需要先定义一个合法的普通变量，也可以得到一个合法的地址。
```

1）新的函数（给一级指针申请空间） （这个函数的基本作用是什么）头文件-->返回值-->形参

​	`① malloc`

```
头文件：#include <stdlib.h>
                       骗子（本质还是一个类型 -- typedef #define）
函数原型：void * malloc(size_t size);   size:大小的意思

返回值类型： void *
void *:
	void:无类型 （作用于函数返回值的时候，如果你的函数不需要返回值）
	*：指针类型
	void * :万能指针 （因为这个函数可以给任意额一级指针变量申请合法的空间）
返回值：
	返回一个万能指针（万能地址），这个地址指向的是一个连续的合法空间。
形参：
	size：设置申请空间的大小(以字节为单位)

实例：
	给一个整形指针变量申请空间：
			申请多大比较正常？ 整形指针变量可以解引用，解引用得到的数据类型是int,4个字节。
	
	第一步：int *p;//野指针
	
	第二步：调用malloc给p申请合法的空间（4个字节）。
			  p = (int *)malloc(4);

	第三步：给合法空间存放上数据。
			
			*p = 12;
	第四步：
			打印数据验证是否存放在合法的空间里面：
			
			printf("%d\n",*p);

	练习：
		 int * p = (int *)malloc(2*sizeof(int));申请了8个字节的空间，在第二个合法空间内			存放数据。

calloc（申请合法空间，有了合法空间就有了合法地址）


free:释放空间
```

`③ calloc函数`

```
头文件：
		#include <stdlib.h>

函数原型：
       void *calloc(size_t nmemb, size_t size);

返回值：
	申请空间成功：返回一个万能指针（万能地址），这个地址指向的是一个连续的合法空间。
	申请空间失败：NULL或者（void *）0
形参一：
		nmemb：内存块的个数
形参二：
		size：每一块的大小
       
```

### **二、二级指针（地址上面还存放了地址(指针)）**

```
① 一级指针变量存放的是普通变量的地址 
② 二级指针变量存放的是一级指针变量的地址 


int a = 10; 假设&a等于0xff33
int *p = &a;假设&p等于 0xff67
int **q = &p;

q 等于多少？0xff67
*q等于多少？*q等于p等于&a等于0xff33
**q等于多少？a

															
练习：使用二级指针变量传参，实现两个普通变量的数据互换。
```

### **三、指针数组、数组指针**

① 概念：

```
指针数组：他是一个数组变量，存放多个指针变量。（里面所有指针变量都要是同一种类型）
数组指针：他是一个指针变量，存放数组变量的地址，这个指针解引用之后就可以得到数组变量。
```
